# 👥 Human Resources Analytics Dashboard

A comprehensive **Human Resources Analytics Dashboard** developed using **Power BI** to help HR professionals monitor workforce performance, employee satisfaction, recruitment metrics, and attrition trends. This project transforms HR data into meaningful insights through interactive dashboards and KPI-driven visualizations, enabling organizations to make informed workforce decisions.

---

## 📌 Project Objective

The primary objective of this project is to analyze HR data and provide actionable insights that improve workforce management, employee retention, and recruitment planning. The dashboard enables HR teams to monitor key performance indicators and identify trends that support strategic decision-making.

---

## 🎯 Key Features

* 📊 Interactive HR Overview Dashboard
* 👥 Employee Demographics Analysis
* 📈 Recruitment & Hiring Analytics
* ⭐ Employee Performance Analysis
* 😊 Employee Satisfaction Insights
* 🔄 Employee Attrition & Retention Analysis
* 📅 Workforce Planning & Forecasting
* 🎛️ Dynamic Filters and Slicers
* 📌 Business KPI Monitoring

* Total Employees

* Attrition Rate
* Average Salary

* Average Job Satisfaction

**Visualizations**

* Employees by Department
* Gender Distribution
* Age Group Analysis
* Job Role Distribution
* Salary by Department
